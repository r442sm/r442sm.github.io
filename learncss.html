目次

CSSとは？

CSSとは、Webページを構成するHTMLに見た目やアニメーションを加えたりするスタイルシート言語です。

基本の構文

CSSにおける基本的な構文はこちらです。


          セレクタ {
          プロパティ: 数値;
          }
単語の意味

セレクタ...CSSを適用させたい要素（buttonタグやh1タグなど）

プロパティ...適用させたいデザイン、アニメーション

数値...適用させるプロパティの数値（大きさや色など）

基本的には、この構文を覚えておくと簡単にCSSを書くことができます！

実際に書いてみる

基本の構文で覚えたことを思い出して実際に書いてみましょう！

今回は例として、inputタグを使ってみます！


          input {
          background-color: #0062ff;
          }
